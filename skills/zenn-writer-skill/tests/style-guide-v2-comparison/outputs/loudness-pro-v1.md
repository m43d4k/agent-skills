## Loudness ProのRealtime AIを用途別に評価する

Loudness ProのRealtime AIを、初見でキック主体の楽曲へ使用した。その後もしばらく操作し、仕事上の用途を検討するために複数の音源を書き出して比較した。

### Realtime AIの処理

Realtime AIはキックを選択的にクリッピングし、一般的なラウドさを得る用途には十分な結果を出した。クリッパーとリミッターの処理配分も正確に感じられた。Loudnessを操作するだけで使えるため、短時間でラウドな結果へ到達できる。

この処理配分を人間が逐次行うのは困難であり、効率的な仕分けという点では高く評価できる。ただし、「正確」「人間には困難」という評価は筆者の試用に基づくものであり、測定によって一般化した結果ではない。また、製品が提案した値は画面に表示されない。

### 表現面の制約

継続して操作すると、処理は効率的で合理的である一方、筆者が目標とする音の表現には到達しなかった。2Mixの段階でクリッピングが設計されていない音源や、クリッパーの表現を避けて作られた音源でピークを均す用途も試した。しかし、複数の書き出しを比較した範囲では、各処理を丹念に設計し、最後にリミッターを1段使った結果の方が仕上がりはよかった。

これは初見時の肯定的な評価を撤回する結果ではない。短時間でラウドにする性能は高いが、AI処理だけで表現まで完成するわけではなく、手作業より常に仕上がりがよいとも判断できない。

### 適用する用途

仕事で使う場合は、Dynamic AIを有効にしたプリリミッターとして利用する。すでに処理を設計した音源で、さらに少しヘッドルームが必要な場合に、AIによる処理配分を利用する方針である。

カジュアルな用途では、クリッパーとリミッターの役割分担を短時間で試す手段になる。Realtime AIの価値は、常に最終品質を高めることではなく、ラウドさへ到達するまでの判断を短縮できる点にある。
