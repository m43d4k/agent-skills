## VX-Qを初期の音作りで使う

VX-Qを、マスタリングではなく受講者が提出した課題用MIXで試した。素材を並べてフェーダーでバランスを取った後の初期段階に使うと、4バンドに限定された構成によって選択時の迷いが減り、低域と高域を通常より速く調整できた。この結果から、初期の音作りを速く進める用途で便利だと評価した。

### 処理を切り替えながら音を組み立てる

ドラムとベースでは、SATモードで同じ性質を与えてまとめる方法と、異なる性質を与えてレイヤーとして分ける方法を切り替えられた。MIXの途中でもモードを切り替えて比較できた。

ローシェルフをブーストしたときは、低域が前へ出るように感じられた。選べる周波数は2種類に限られるものの、MIXの土台となる低域を作りやすい。さらにEVENを個別に調整してLPFを使うと、整理されながらレンジの広い結果が得られたため、ローシェルフによる低域処理をVX-Qの特に優れた特徴と見ている。

MODE SとZでは、音を飽和させて狭くなったレンジをEQで広げることで、刺激を強くせずに力のある音を作れた。複数のエレキギターや、反対側へ配置するダビングでは、歪みを奥へ置いて表現を分ける用途でも機能した。筆者がこの2つのモードを好むのは、これらの結果に基づく。

### 制約と未確認事項

ディエッサー相当の機能がないため、さらに強いEQ処理を行う余地には制約がある。VARIは音像が傾いて聞こえる問題が強く、デメリットと判断した。

MODE Zの名称はZenerに由来すると理解しているが、MODE Sの由来は確認できていない。Steelではないかという推測は、確認済みの事実として扱わない。また、低域が前へ出る感覚や各モードの評価は、今回試した素材の範囲に限られる。
