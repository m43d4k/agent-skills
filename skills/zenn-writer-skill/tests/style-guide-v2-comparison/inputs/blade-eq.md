# Blade EQ 共通入力

## 出典

- `samples/moroishimasaoki/2026-06-28-cedar-blade-eq.md`
- 使用範囲: 148〜173行

## 使用条件と試行

- Blade EQのブーストとカットを実際に試した。
- 使用前は、SonicEQに近い音ではないかと予想していた。

## 観測

- 完全に無色ではなく、筆者にはTPDFディザーを加えたときに似た、わずかな滑らかさが感じられた。
- 特定の高域寄り、低域寄りという傾向は感じられなかった。
- 筆者には、フラット寄りでありながら、強拍が明瞭になり、弱拍を聞き取りやすくする性質が感じられた。
- 使用前に予想したSonicEQよりも、TC Electronic PEQ3000を想起させる音だった。
- ブーストでは無理なく音を手前へ出せた。
- カットは自然で、設定値に対応した結果を得られた。
- 比較対象のKirchhoff-EQでは、処理後に滑らかな質感が残り、それが上品に感じられる場合と、冗長に感じられる場合があった。
- Blade EQはKirchhoff-EQほど無色ではないが、筆者の試用範囲では素材を選ぶ感覚がなかった。

## 判断と判断履歴

- 実際の試用により、SonicEQに近いという使用前の予想から、PEQ3000に近いという評価へ変わった。
- ブーストとカットの結果、および素材を選びにくい音の傾向から、メインEQの候補として実用的だと判断した。

## 留保

- 音の滑らかさや比較対象との違いは、測定値で示した結果ではなく、筆者の聴感上の評価である。
- ハードウェアDSPを使った経験の有無によって、音の捉え方が変わる可能性がある。
- 「PEQ3000を想起させた」は類似性についての評価であり、同じ設計や同じ音であることを意味しない。
