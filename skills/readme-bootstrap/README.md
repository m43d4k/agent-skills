# readme-bootstrap

新規リポジトリ向けに、最低限の日本語READMEと`LICENSE`ファイルを作成する。

後から内容を追加できる初期状態を作る。

## 作成内容

- `README.md`: タイトル、概要、作者、ライセンスを記載する
- `LICENSE`: 同梱のテンプレートからMIT Licenseを作成する

READMEの本文と見出しは日本語の常体で記述する。既存の`LICENSE`は上書きしない。

## 適用対象

`README.md`が存在しない、新規作成直後のリポジトリを対象とする。

## 使い方

AIエージェントに`readme-bootstrap`スキルを指定し、READMEの作成を依頼する。

```text
readme-bootstrapスキルを使い、README.mdを作成する。
```
