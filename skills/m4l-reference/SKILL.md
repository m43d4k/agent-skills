# Max for Liveの参照情報

この`SKILL.md`と同じディレクトリにある`settings.toml`を読む。

`settings.toml`の`guidelines_dir`が示すディレクトリ内の指示を読み、Max for Live作業へ適用する。

利用できる参照資料は次のとおり。

- `playbooks_dir`が示すディレクトリ内の、ユーザー固有の実践情報
- 対象Live内蔵Maxに付属するObject Reference、User Guide、help、Examples
- `sources_dir`が示すディレクトリ内の、公式Web資料のURL、用途、バージョン上の注意

参照資料には固定の優先順位を設けない。

互換性は対象Liveと、そのLiveに内蔵されたMaxを基準とする。スタンドアロンMax、Liveで選択された外部Max、別バージョンを暗黙に代用しない。

確認できない内容を仕様として断定しない。通常作業中にguidelines、playbooks、sourcesまたはこのSkillを変更しない。
