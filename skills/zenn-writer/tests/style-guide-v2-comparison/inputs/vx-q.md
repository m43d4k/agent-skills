# VX-Q 共通入力

## 出典

- `samples/moroishimasaoki/2025-12-29-kiive-audio-vx-q.md`
- 使用範囲: 230〜275行

## 使用条件と試行

- マスタリングではなく、受講者が提出した課題用MIXで試した。
- 素材を並べてフェーダーでバランスを取った後の、初期の音作りに使用した。
- ドラムとベースの音作り、および複数のエレキギターの処理にも使用した。
- 使用前の予想や評価は入力材料にない。

## 観測

- 4バンドに限定された構成により、選択時の迷いが少なく、低域と高域のバランスを通常より速く調整できた。
- SATモードでは、ドラムとベースに同じ性質を与えてまとめる方法と、異なる性質を与えてレイヤーとして分ける方法を切り替えられた。
- MIXの途中でもSATモードを切り替えて比較できた。
- ローシェルフをブーストすると、筆者には低域が前へ出るように感じられた。
- ローシェルフで選べる周波数は2種類に限られる。
- EVENを個別に調整してLPFを積極的に使うと、整理されながらレンジの広い結果が得られた。
- MODE SとZは、音を飽和させて狭くなったレンジをEQで広げる使い方により、刺激を強くせずに力のある音を作れた。
- MODE SとZは、複数のエレキギターや反対側へ配置するダビングの歪みを奥へ置き、表現を分ける用途でも機能した。
- VARIは、筆者には音像が傾いて聞こえる問題が強く感じられた。

## 判断

- 初期の音作りを速く進める用途で便利だと評価した。
- ローシェルフによる低域処理を、特に優れた特徴として評価した。
- MODE SとZの表現を筆者は好んだ。

## 制約と未確認事項

- ディエッサー相当の機能があれば、さらに強いEQ処理をしやすいと考えたが、その機能はない。
- VARIの音像が傾いて聞こえる点はデメリットと判断した。
- MODE Zの名称はZenerに由来すると理解している。
- MODE Sの由来は未確認である。Steelではないかという推測はあるが、確認済みの事実ではない。
- 低域が前へ出る感覚や各モードの評価は、筆者が試した素材に基づく判断である。
