## VX-Qを初期の音作りで使う

VX-Qを、マスタリングではなく受講者が提出した課題用MIXで試した。素材を並べてフェーダーでバランスを取った後の初期段階に使用し、ドラム、ベース、複数のエレキギターを処理した。

### 4バンド構成とSATモード

4バンドに限定された構成は選択時の迷いが少なく、低域と高域のバランスを通常より速く調整できた。SATモードでは、ドラムとベースに同じ性質を与えてまとめる方法と、異なる性質を与えてレイヤーとして分ける方法を選べる。MIXの途中でもモードを切り替えて比較できるため、初期の音作りを速く進める用途で便利だった。

### 低域とMODE S、Zの処理

ローシェルフをブーストすると、低域が前へ出るように感じられた。選べる周波数は2種類に限られるが、MIXの土台となる低域を作りやすい。EVENを個別に調整し、LPFを積極的に使うと、整理されながらレンジの広い結果が得られた。このローシェルフによる低域処理は、VX-Qの特に優れた特徴だと評価した。

MODE SとZでは、音を飽和させて狭くなったレンジをEQで広げることにより、刺激を強くせずに力のある音を作れた。複数のエレキギターや、反対側へ配置するダビングの歪みを奥へ置き、表現を分ける用途でも機能した。筆者はこの2つのモードの表現を好んでいる。

### 制約と未確認事項

ディエッサー相当の機能がないため、さらに強いEQ処理を行う余地には制約がある。VARIは音像が傾いて聞こえる問題が強く、デメリットと判断した。

MODE Zの名称はZenerに由来すると理解している。一方、MODE Sの由来は確認できていない。Steelではないかという推測はあるが、事実としては扱えない。低域が前へ出る感覚や各モードの評価も、今回試した素材に基づく判断である。
