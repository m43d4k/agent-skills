# README Writer

リポジトリ内の実装や設定、既存ドキュメントを調査し、日本語READMEの作成と修正を支援するAgent Skill。

確認できない情報を推測で補わず、対象リポジトリ固有の内容をREADMEへ反映する。

## 対象

- 日本語READMEの新規作成
- 既存READMEの修正、追記、構成変更

## 方針

- 実際の動作を規定するソース、設定、スクリプト、CI、既存ドキュメントを調査する
- 利用者が何をするためのリポジトリかを、実装方式より優先して説明する
- 既存READMEの用語、見出し階層、説明順序を尊重する
- 本文と見出しを日本語の常体で記述する
- 未実装の機能や確認できないコマンド、動作、対応環境を追加しない
- 宣伝的な表現や、リポジトリと関係のない一般的な説明を追加しない

詳細な規則は[`SKILL.md`](SKILL.md)に記載している。

## 使い方

AIエージェントに`readme-writer`スキルを指定し、対象のREADMEと依頼内容を伝える。

```text
readme-writerスキルを使い、README.mdを修正する。
```
